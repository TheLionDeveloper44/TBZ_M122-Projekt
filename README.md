# Einleitung
Persönliches Projekt für die Technische Berufsschule Zürich, Modul 122. Dieses Projekt beinhaltet Inhalte für das zweite Block (Projektteil).

# Was ist dieses Projekt
Vielleicht kennt man Ninite oder Chocolatery. Das sind Softwares die dem Benutzer oder Unternehmen anbieten, Apps automatisch und einfach zu installieren, etwa so ähnlich wie eine MDM verwaltung. Der unterschied zwischen einer MDM und so etwasem wie Ninite ist es, dass es keine grosse Verwaltungen gebraucht werden. Ein Privaterbenutzer kann sich einfach Apps aussuchen und anschliessend mit einer .exe automatisch installieren lassen, ohne grosser Programmierung und Aufwand.

Es gibt aber einen Problem. Ninite hat nur gewisse Apps zur Verfügung und so etwas wie Chocolatery kostet für einen Privatbenutzer.

Dieses Projekt ist etwas was ich schon machen wollte für meinen Privatgebrauch. Ich wollte eine mischung zwischen der kostenlosen Variante und der kostenpflichtigen Variante haben, wo ich selber Apps auswählen kann, die zur Verfügung sind und ich sie mit einem Klick in einem GUI installieren kann.
